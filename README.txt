Rokuwiki Template
=====

Robin Lab's template adapted from default Dokuwiki template.

Put under lib/tpl/

Deploy plugin icons
-----

Wrap plugin:

    ln -s lib/tpl/rokuwiwi/wrap-plugin/images/note lib/plugins/wrap/images/note
